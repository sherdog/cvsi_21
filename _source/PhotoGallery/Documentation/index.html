Installation

2.3x Version

1. Connect to you server via FTP or log into cPanel file manager. Open OpenCart root folder, upload and extract the Gallery(version).zip file into it.
2. With that done add /install to your site link, e.g. http://yourdomain.com/install. Click the Continue button on the extension installation page.
3. Next, install extensions: unzip the Gallery(version).zip file and navigate to the Extensions -> Extensions Installer tab in OpenCart admin panel.
4. Upload files from the Gallery folder one by one using the Upload button.

5. When all the extensions are uploaded, activate them via Extensions -> Modifications, click the Refresh button in the top right corner.

6.Assign Permission to your respected user group.


7. An Gallery icons display on left side. 8. Go to Extension > Modules .click install button to install module


9. Now Gallery is installed. Congratulations!

If some reason the Extension installer does not work for you, you can try the old fashioned way.
Read below:

1. Unzip the downloaded ZIP file into a new folder before you copy the files. If your OpenCart is not a fresh installation, files backup is highly recommended.

2. Copy all files from /upload folder over your current OpenCart installation preserving the directory structure.

3. Install Ocmod using the Upload button in admin panel.
4. When ocmod upload , activate this form Extensions > Modifications.
5. Assign Permission to your respected user group.


6. An Gallery icons display on left side. 7. Go to Extension > Module .click install button to install module

7. Now Gallery is installed. Congratulations!
